જયભારત સહ ઉપરોક્ત વિષયમાં જણાવવાનું કે, મુ. {{ village }} ખાતે આપના કબજા હેઠળની……………આરે જમીનમાં પરચુરણ વસુલાત / મહેસુલ બાકી રૂ|. {{ total }} અંકે રૂ|. {{ totalWord }} પુરા આજદિન સુધીમાં ભરપાઈ કરેલ નથી, આ અંગે તલાટી કમ મંત્રી દ્વારા આપને વિગતવાર માંગણું પાઠવી મહેસુલ બાકી વસુલાત ભરપાઈ કરવા જાણ કરવામાં આવેલ છે. તેમ છતાં તમારા દ્વારા મહેસુલ ભરપાઈ ન થતા સંદર્ભ ૧ થી બોમ્બે લેન્ડ રેવન્યુ એક્ટ-૧૮૭૯ ની કલમ ૧૫૨ હેઠળ આપને નોટીસ આપવામાં આવેલ છે. તેની પણ આપે અવગણના કરી ઈરાદાપૂર્વક મહેસુલ ન ભરવાના વલણને વળગી રહી તમે જમીન મહેસુલ કાયદા હેઠળ મહેસુલ ભરવાની તમારી કાયદેસર કલમ ૧૪૮ મુજબ આપનું કૃત્ય ચોથાઈ દંડને પાત્ર કૃત્ય બનતું હોય, રૂ. {{ total }} નો ચોથાઈ દંડ રૂ. {{ quarter }} આપની પાસેથી શા માટે વસુલ ન કરવા અને પરચુરણ વસુલાત બાકી રકમ {{ quarter }} અંકે રૂ. {{ quarterWord }} પુરા મહેસુલ બાકી તરીકે મહેસુલ કાયદા મુજબ કાયદેસરની કાર્યવાહી કરી શા માટે વસુલ ન કરવા તે અંગે આપ કોઈ રજૂઆત કરવા ઇચ્છતા હોય તો તા {{ date }} ના રોજ {{ time }} કલાકે તાલુકા પંચાયત કચેરી, {{ village }} ખાતે આપને જાતે હાજર રહી રજૂઆત કરવા આપને તક આપવામાં આવે છે.
      જણાવેલ સ્થળે સમય મુજબ રજૂઆત કરવામાં નહિ આવે તો આપ આ અંગે કોઈ રજૂઆત કરવા માંગતા નથી. એમ માની ગુણદોષ મુજબ કાયદેસર નિર્ણય લેવામાં આવશે જેની નોધ લેશો.
